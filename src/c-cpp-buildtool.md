# C/C++ 開発環境の構築

## コンパイラ

### Windows で使用できるコンパイラ (商用利用)

| コンパイラ   | OSS のビルド | 内製コードのビルド |
| ------------ | :----------: | :----------------: |
| MSVC Pro     |      O       |         O          |
| MSVC         |      O       |         X          |
| MINGW (GCC)  |      O       |         O          |
| Intel OneAPI |      O       |         O          |

O: ライセンスの問題なし
X: ライセンスの問題あり
